## Installation and Setup Instructions

Clone down this repository. You will need node and npm installed globally on your machine.

To run the backend:

cd server

Installation:

npm install

To Start Server:

npm start

----

To run the frontend:

cd web

Installation:

npm install

To Start Server:

npm start
